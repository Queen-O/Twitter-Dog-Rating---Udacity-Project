Analysis on WeRateData Twitter Udacity Project¶

Introduction¶

This is an analysis carried out on three datasets merged as one (twitter_archive_master.csv). These datasets centered on dog rating include an archived WeRateDogs tweets in csv format, image prediction carried out on the archive dataset in tsv format and an API json data querried with the tweets IDs. These data were reasonably cleaned and merged and saved as twitter_archive_master.csv

Insights from Data¶

After the assessment and cleaning has been done successfully, insights were gotten from the data based on visualizations.

  1. The trend of retweets and favorites increased to an all time high in 2016 and since then, favorite has been at the same till 2017 While retweets declined a bit towards 2017. Note that the data used for this project was collected between 2015 and 2017.
